Latin Word Order and Case – Practice for Class 6

This page gives extra practice with:

  • Latin word order and case endings (nominative vs. accusative)
  • Matching Latin patterns you have seen in Class 6 slides and AMT Chapters 2–3, 6
  • Recognizing how anatomical Latin phrases in Terminologia Anatomica are built

Use this together with the textbook and workbook exercises on Chapters 2–3 and 6.[file:6][file:5][file:8]

1. Who Sees Whom? (Musculus & Nervus)

Each sentence uses the same three words (musculus, nervus, videt) with different endings.[file:8] Identify the subject (nominative), the object (accusative), and give a good English translation.

  1. musculus nervum videt
  2. nervus musculum videt
  3. musculum nervus videt
  4. videt musculum nervus

Show / hide suggestions

  1. musculus = subject, nervum = object → “The muscle sees the nerve.”
  2. nervus = subject, musculum = object → “The nerve sees the muscle.”
  3. Same roles as (2), even though the order is different.
  4. Same roles as (2): endings, not position, decide who does what.[file:8]

2. New Anatomical Nouns, Same Pattern

Now use other nominative/accusative pairs that occur in AMT and the slides:[file:6][file:8]

  • bronchus / bronchum – the bronchus
  • pulmo / pulmonem – the lung
  • arteria / arteri-am – the artery (feminine)

For each Latin sentence, underline the subject, circle the object, and translate.

  1. bronchus pulmonem videt
  2. pulmo bronchum videt
  3. arteria pulmonem videt
  4. pulmonem arteria videt

Show / hide suggestions

  1. bronchus = subject → “The bronchus sees the lung.”
  2. pulmo = subject → “The lung sees the bronchus.”
  3. arteria (nominative) = subject → “The artery sees the lung.”
  4. Same meaning as (3): word order changes, endings do not.[file:6]

3. Matching Latin Phrases to English

All of the following imitate patterns from AMT’s general anatomy and respiratory chapters.[file:6] For each Latin phrase, (a) identify the main noun, (b) decide if it is singular or plural, and (c) give a natural English phrase.

  1. musculus thoracicus
  2. musculi thoracici
  3. nervus brachialis
  4. nervi brachiales
  5. vasa pulmonalia
  6. pleurae parietales
  7. septa nasalia

Show / hide suggestions

  1. “the thoracic muscle” / “the muscle p.t. the chest” (singular).[file:6]
  2. “the thoracic muscles” / “the muscles p.t. the chest” (plural).[file:6]
  3. “the brachial nerve” / “the nerve p.t. the arm”.[file:6]
  4. “the brachial nerves” / “the nerves p.t. the arm(s)”.[file:6]
  5. “the vessels p.t. the lungs” (neuter plural with ‑alia).[file:6]
  6. “the parietal pleurae”.[file:6]
  7. “the nasal septa” / “the septa of the nose”.[file:6]
